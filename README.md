# Reproducibility Repository for the PhD Thesis of Janette Suherli (2026)

This repository is intended as an archival reproducibility resource accompanying the PhD thesis and associated publications. 

"Probing the Environments of Central Compact Objects in Supernova Remnants with Optical Integral Field Spectroscopy"


The repository contains analysis scripts, figure-generation or visualization scripts, and modelling utilities. 
The work focuses on optical integral field spectroscopy (IFS) studies of central compact object (CCO) environments in young supernova remnants (SNRs), primarily using observations from the Multi Unit Spectroscopic Explorer (MUSE) on the ESO Very Large Telescope (VLT).
